CompleteWhen(IDataflowBlock, IEnumerable<Task>)

Completes the block when all of the specified tasks are completed.

Declaration
public static void CompleteWhen(this IDataflowBlock target, IEnumerable<Task> tasks)
Parameters
Type Name Description
IDataflowBlock target
IEnumerable<Task> tasks
Remarks

If any task faults, Fault(Exception) of the target will be called soon. If you want to wait for the all tasks even if any task faults, specify Task.WhenAll(tasks) to tasks.

Examples

This sample shows how to merge some source blocks.

IReadOnlyList<ISourceBlock<T>> sources;
ITargetBlock<T> target;

// Link sources to target
foreach (var source in sources)
    source.LinkTo(target);

// Propagate completion
target.CompleteWhen(sources.Select(s => s.Completion));
Exceptions
Type Condition
ArgumentNullException

target or tasks is null.
