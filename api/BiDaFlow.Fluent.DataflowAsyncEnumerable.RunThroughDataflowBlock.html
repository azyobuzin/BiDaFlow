RunThroughDataflowBlock<TInput, TOutput>(IAsyncEnumerable<TInput>, Func<CancellationToken, IPropagatorBlock<TInput, TOutput>>)

Creates an IAsyncEnumerable<T> that transforms the elements from source with IPropagatorBlock<TInput,TOutput>.

Declaration
public static IAsyncEnumerable<TOutput> RunThroughDataflowBlock<TInput, TOutput>(this IAsyncEnumerable<TInput> source, Func<CancellationToken, IPropagatorBlock<TInput, TOutput>> propagatorFactory)
Parameters
Type Name Description
IAsyncEnumerable<TInput> source

A sequence of elements that will be transformed.
Func<CancellationToken, IPropagatorBlock<TInput, TOutput>> propagatorFactory

A function that creates IPropagatorBlock<TInput,TOutput> used to transform the elements.

This function is called each time GetAsyncEnumerator(CancellationToken) is called.
Returns
Type Description
IAsyncEnumerable<TOutput>
Type Parameters
Name Description
TInput

The type of the elements in the source sequence.
TOutput

The type of the elements in the result sequence
Remarks

If the dataflow block instantiated in propagatorFactory will not be used somewhere else, you can set true to SingleProducerConstrained of the options object for the block.

Examples
await AsyncEnumerable.Range(1, 100)
    // Process elements in parallel with IPropagatorBlock
    .RunThroughDataflowBlock(cancellationToken =>
        new TransformBlock<int, int>(
            x => x * 10,
            new ExecutionDataflowBlockOptions
            {
                CancellationToken = cancellationToken,
                BoundedCapacity = 6,
                MaxDegreeOfParallelism = 4,
                SingleProducerConstrained = true,
            })
    )
    // Subsequent process can be written with System.Linq.Async
    .ForEachAsync(x => Console.WriteLine(x));
Exceptions
Type Condition
ArgumentNullException

source or propagatorFactory is null.
