AsAsyncEnumerable<TOutput>(ISourceBlock<TOutput>)

Creates an IAsyncEnumerable<T> to consume the items from ISourceBlock<TOutput>.

TOutput

Specifies the type of data contained in the source.
Remarks

The returned IAsyncEnumerable<T> behaves following:

  1. Makes a link from source to the internal target block when GetAsyncEnumerator(CancellationToken) is called.
  2. Consumes an element of source when MoveNextAsync() is called.

Offers received when the user is not awaiting MoveNextAsync() will be postponed because the internal target block doesn't have a buffer.

Exceptions
Type Condition
